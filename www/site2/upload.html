Upload a file to the server

Use the form below to upload a file to /upload. This page uses a plain HTML form with multipart/form-data so browsers will perform a standard POST without any JavaScript. If your browser supports JavaScript, you can still use the form normally — no client-side script is required.

Select a file then click Upload — the browser will send it to /upload.
